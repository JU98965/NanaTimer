## 개요
계획 없이 공부 시간을 관리해 보실래요?

괜찮아요! 계획이 없는 게 계획이니까요!

"나나타이머"는 7일 동안 틈틈이 공부하며 목표 시간을 달성하는 공부 타이머입니다.

<br/>

## 사용 기술
**디자인 패턴은 MVVM을 사용했습니다.**

| 이름 | 목적 |
| --- | --- |
| SwiftUI | 레이아웃을 구성하고 반응형 프로그래밍을 구현합니다. |

<br/>

## 코드 컨벤션
- 더 이상 상속되지 않는 클래스는 final 키워드를 사용합니다.
